# Interact2Hack 2026 - Datamine

## Descripción
Este proyecto tiene como base resolver el problema propuesto por deuna!, construir un modelo predictivo donde se debe identificar el alto riesgo de abandono en los próximos 30 días. 

## Objetivos
-Entregar una herramienta accionable para el equipo comercial.
-Presentar una solución completa con modelo predictivo de churn, explicación de las variables relevantes, dashboard, plan de acción y presentación final.

## Alcance
Incluye un procesamiento y análisis de datos, entrenamiento y evaluación de modelos, explicabilidad, construccion de dashboard y recomendaciones de negocio.
